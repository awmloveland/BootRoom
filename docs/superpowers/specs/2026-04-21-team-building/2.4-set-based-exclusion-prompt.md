## How to approach it

1. Open `lib/autoPick.ts`. Inside `autoPick`, before the pair-pinning loop, declare `const excluded = new Set<Player>()`.
2. In each branch of the loop, replace `searchPool = searchPool.filter((p) => ...)` with `excluded.add(...)` calls — add the guest, and in the "associated found in pool" case add the assoc too.
